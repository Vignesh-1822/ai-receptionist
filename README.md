# Aria — AI Booking Agent

An AI-powered voice booking agent. Users speak naturally to Aria, 
who checks availability and confirms appointments in under 60 seconds.

## Demo
[Live URL] | [Demo Video]

## How it works
1. User clicks mic on the web app
2. Aria (powered by Claude) handles the conversation
3. Booking confirmed → calendar invite generated

## Tech Stack
- **Frontend**: Next.js, TailwindCSS, Retell Web SDK
- **Backend**: FastAPI, Python
- **Voice**: Retell AI
- **LLM**: Claude (Anthropic)
- **Database**: Supabase
- **Deploy**: Vercel (frontend) + Railway (backend)
